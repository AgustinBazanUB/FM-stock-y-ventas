# Botonera Bluetooth

La botonera se trata como un teclado físico; la app no usa Web Bluetooth.

## Conectar y usar

1. Emparejá la botonera desde Bluetooth en Android o iPhone.
2. Abrí la app en Google Chrome e ingresá como vendedor.
3. Entrá en **Nueva venta**: la captura se activa automáticamente.
4. Las teclas de productos agregan una unidad. `Enter` registra, `Backspace` quita la última unidad, `+` suma y `-` resta el último producto.

Para asignar una tecla: como administrador, editá el producto, tocá **Grabar tecla** y presioná la tecla física. Después agregá o volvé a guardar ese producto en el stock de cada ubicación.

## Si no responde

- Confirmá que Bluetooth siga conectado.
- Tocá **Reactivar botonera** si el navegador perdió el foco.
- Revisá Num Lock.
- Probá el modo Android/iOS de la botonera.
- Verificá que Chrome esté activo y que no haya un campo de texto o modal abierto.
- Algunas botoneras envían códigos distintos según el sistema; volvé a grabar la tecla desde el mismo celular que se usará para vender.
